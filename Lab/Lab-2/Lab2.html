Lab #2


Description of Lab #2

1. Create an HTML5 home page that displays the course title, my name, and date (in the footer), also, provide links to a Lab #2 page and a site map page in the left column.
2. Create a Lab #2 page that uses footer, details and mark tags, which are new features to HTML5, with a Lab #2 header and a brief description of the lab.
3. Create a site map page that has two links: one to home page and one to the Lab #2 page.
4. Create a CSS style sheet and add links to it on each of the html pages. Added styles that make the site look reasonably professional.
5. Add a JavaScript file to apply a theme to each page that contains an html div element with a class whose name is either lighttheme or darktheme. Call this function from the body load event.
6. Add lighttheme and darktheme styles that each set the CSS style properties: background-color and color.
7. On the home page add lighttheme div, on the Lab #2 and site map pages add darktheme div.
8. At the top of every page, add three check boxes for light theme, dark theme, page theme.
    a) The first two will be used to change the theme, ignoring the divs with theme classes. So all pages will have the same theme.
    b) By default, each page will have its own theme and the "Page Theme" checkbox will be selected if none of the first two is checked.
    c) Save the state of these buttons in local storage, and retrieve from local storage on page load.
    d) If select "Page Theme" on the page, you will need to refresh the page manually to see the changes.
